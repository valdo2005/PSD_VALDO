Exploratory Data Analysis (EDA) database MySQL & PostgreSQL menggunakan Power BI#

Judul Dataset#

Saya menggunakan dataset Iris yang populer dalam pembelajaran mesin. Dataset ini sering digunakan untuk demonstrasi dan latihan algoritma klasifikasi karena strukturnya yang bersih dan terstruktur, memungkinkan pemahaman tentang pemodelan dan evaluasi model pembelajaran mesin.

Dataset ini terdiri dari 150 baris atau sampel dengan 4 atribut yaitu:

  • Sepal length

  • Sepal width

  • Petal length

  • Petel width

Serta memiliki satu atribut class yang berisi 3 sepsiies yaitu: (Setosa, Versicolor, dan Virginica)

Tujuan Proyek#

  1. Membuat database Iris di MySQL dan PostgreSQL.

  2. Menghubungkan database tersebut dengan Power BI.

  3. Melakukan analisis deskriptif berupa min, max, dan rata-rata setiap atribut.

  4. Menampilkan distribusi jumlah data setiap kelas dalam bentuk grafik batang.

Peoses Menghubungkan dengan DBeaver:#

  • Buka DBeaver → New Database Connection → pilih MySQL/Postgrasql.

  • Masukkan hostname, port, username, dan password(mysql dan postgresql).postgresql juga bisa menggunakan url seperti yang sudah dilampirkan pada informasi koneksi diatas

  • Buat database baru iris_db.

  • Buat tabel dengan struktur sesuai dataset yaitu data iris.

Setelah menghubungkan atau melakukan koneksi di DBeaver, selanjutnya kita melakukan Integrasi Database ke Power BI

Integrasi Database ke Powe BI#

  • Buka Power BI Desktop.

  • Pilih Get Data → MySQL Database atau PostgreSQL Database.

  • Masukkan hostname, port, username, dan password sesuai konfigurasi.

  • Pilih database iris_db dan tabel iris.

  • Data siap divisualisasikan.

Deskripsi Data:#

Interpretasi Warna Batang#

  • Petallength (biru muda): dominan pada Iris-virginica dan Iris-versicolor, tetapi kecil pada Iris-setosa.

  • Petalwidth (biru tua): relatif kecil kontribusinya pada semua kelas, namun masih terlihat.

  • Sepallength (oranye): besar pada Iris-virginica dan Iris-versicolor, sedang pada Iris-setosa.

  • Sepalwidth (ungu): cukup menonjol pada Iris-setosa, menandakan bahwa meskipun ukuran petal kecil, sepal lebih lebar.

Interpretasi Tabel#

  1. Iris-virginica memiliki nilai total tertinggi karena petallength dan sepallength yang besar → sesuai dengan karakteristik biologis bunga yang lebih panjang.

Hubungan Grafik Batang#

  • Warna oranye (Sepallength) dan biru muda (Petallength) adalah kontributor utama perbedaan antar kelas.

  • Setosa unik karena Sepalwidth (ungu) justru mendominasi dibandingkan panjang petalnya → menunjukkan perbedaan morfologi yang jelas.
